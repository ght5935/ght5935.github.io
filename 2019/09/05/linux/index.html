linux 判断文件夹或文件是否存在
linux
发布日期:   2019-09-05
文章字数:   103
阅读时长:   1 分
阅读次数:  

文件夹不存在则创建

if [ ! -d "/data/" ];then
mkdir /data
else
echo "文件夹已经存在"
fi

文件存在则删除

if [ ! -f "/data/filename" ];then
echo "文件不存在"
else
rm -d /data/filename
fi

判断文件夹是否存在

if [ -d "/data/" ];then
echo "文件夹存在"
else
echo "文件夹不存在"
fi

判断文件是否存在

if [ -f "/data/filename" ];then
echo "文件存在"
else
echo "文件不存在"
fi
